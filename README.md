# 仓库管理系统微信小程序

这是一个用于库存管理的微信小程序示例，采用微信小程序原生 `WXML + WXSS + JS` 实现，数据默认存储在本地缓存中，便于直接导入微信开发者工具预览。

## 功能

- 首页仪表盘：总库存数量、库存总金额、库存预警、商品总数、仓库数量、渠道商剩余款项合计。
- 渠道商款项：支持普通渠道商、3W品牌渠道商、台州伊文渠道商分类统计。
- 库存数量：支持新增、编辑、删除商品信息，维护在库数量、登记单价与预警线。
- 仓库数量：支持新增仓库，并自动统计每个仓库当前库存数量。
- 出入库流水：支持出库记录、入库记录、退货入库记录，并自动回写商品库存。

## 使用方式

1. 打开微信开发者工具。
2. 选择“导入项目”，目录选择本仓库根目录。
3. AppID 可使用测试号或 `touristappid`。
4. 编译后进入首页即可使用。
