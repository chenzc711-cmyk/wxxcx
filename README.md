# 仓库管理系统微信小程序

这是一个用于库存管理的微信小程序示例，采用微信小程序原生 `WXML + WXSS + JS` 实现，数据默认存储在本地缓存中，便于直接导入微信开发者工具预览。

## 功能

- 首页仪表盘：总库存数量、库存总金额、库存预警、商品总数、渠道商剩余款项合计。
- 仪表盘出入库统计：今日入库/出库数量与金额、本月入库/出库数量与金额、本月累计出入库数量与金额。
- 渠道商款项：支持普通渠道商、3W品牌渠道商、台州伊文渠道商分类统计。
- 库存数量：支持新增、编辑、删除商品信息，维护商品规格、商品分类、在库数量、登记单价与预警线。
- 入库记录：登记入库日期、商品名称、供应商、数量、单价、备注，并自动增加库存。
- 出库记录：登记出库日期、商品名称、商品规格、数量，并自动扣减库存。
- 退货入库记录：登记退货日期、订单号、物流公司、物流单号、商品名称、商品规格、数量、操作人、备注，并自动增加库存。
- 所有包含日期的记录界面均支持开始日期与结束日期筛选。

## 使用方式

1. 打开微信开发者工具。
2. 选择“导入项目”，目录选择本仓库根目录。
3. AppID 可使用测试号或 `touristappid`。
4. 编译后进入首页即可使用。
